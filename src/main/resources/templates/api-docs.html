SecureBank REST API
Complete REST API for SecureBank. All endpoints return JSON with a consistent response wrapper. Use session-based authentication.
Base URL http://localhost:8080/api/v1
📦 Standard Response Format
All responses follow this structure:
{ "success": true, "message": "Success", "status": 200, "timestamp": "2024-01-15T10:30:00", "data": { ... } }
🔐 Authentication
POST /auth/register Register new customer PUBLIC ▼
Request Body
{ "fullName": "John Doe", "email": "john@example.com", "password": "MyPass@123", "confirmPassword": "MyPass@123", "phone": "9876543210", "address": "123 Main Street, Pune" }
Response 200
{ "success": true, "message": "Registration successful!", "data": "john@example.com", "status": 200 }
POST /auth/login Authenticate user PUBLIC ▼
Request Body
{ "email": "customer@securebank.com", "password": "Customer@123" }
Response 200
{ "success": true, "message": "Login successful", "data": { "email": "customer@securebank.com", "status": "authenticated" } }
GET /auth/me Get current user info AUTH REQUIRED ▼
Response 200
{ "success": true, "data": { "id": 1, "fullName": "John Doe", "email": "customer@securebank.com", "role": "CUSTOMER", "active": true, "locked": false } }
GET /auth/dashboard Full dashboard data AUTH REQUIRED ▼
Response 200
{ "success": true, "data": { "fullName": "John Doe", "email": "customer@securebank.com", "totalBalance": 50000.00, "totalAccounts": 1, "totalTransactions": 5, "pendingLoans": 0, "accounts": [...] } }
🏦 Accounts
GET /accounts Get all my accounts AUTH REQUIRED ▼
Response 200
{ "success": true, "data": [ { "id": 1, "accountNumber": "SB1234567890", "accountType": "SAVINGS", "balance": 50000.00, "active": true, "ownerName": "John Doe", "ownerEmail": "john@example.com" } ] }
GET /accounts/balance Get total balance AUTH REQUIRED ▼
Response 200
{ "success": true, "message": "Total balance retrieved", "data": 50000.00 }
💳 Transactions
GET /transactions Transaction history AUTH REQUIRED ▼
Param Type Required Description
accountId Long No Filter by account ID
Example: GET /transactions?accountId=1
POST /transactions/transfer Fund transfer AUTH REQUIRED ▼
Request Body
{ "toAccountNumber": "SB9876543210", "amount": 5000.00, "description": "Rent payment" }
Response 200
{ "success": true, "message": "Transfer successful! Ref: TXN...", "data": "TXNABC123DEF456" }
🏛️ Loans
GET /loans Get my loan applications AUTH REQUIRED ▼
Response 200
{ "success": true, "message": "1 loans found", "data": [ { "id": 1, "amount": 50000.00, "tenureMonths": 24, "interestRate": 10.5, "status": "PENDING", "appliedAt": "2024-01-15T10:30:00" } ] }
POST /loans/apply Apply for loan AUTH REQUIRED ▼
Request Body
{ "amount": 50000.00, "tenureMonths": 24 }
Response 200
{ "success": true, "message": "Loan application submitted!", "data": "Loan #1" }
